# Elesh Vaishnav - 3D Portfolio

Interactive 3D portfolio built with Three.js, featuring rotating 3D geometry with particle effects.

## Setup

```bash
npm install
npm run dev    # Development server
npm run build  # Production build
npm run start  # Production server
```

## Features

- **3D Rotating Cube** with wireframe overlay
- **Particle System** - Floating points in space
- **Mouse Interaction** - Cube follows mouse movement
- **Scroll Zoom** - Zoom in/out with mouse wheel
- **Terminal Aesthetic** - Green matrix-style design
- **Project Info** - DeepClaw & Parmana showcase

## Projects

- **DeepClaw**: Agentic AI system
- **Parmana**: Local AI assistant with Telegram integration

## Controls

- Drag mouse to rotate
- Scroll to zoom
- Click buttons for quick actions
